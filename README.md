 Introducción
El proyecto Control de Accesos del Conjunto Residencial Los Robles surge como respuesta a la necesidad de mejorar la seguridad y optimizar los procesos internos del conjunto.  
Antes de la implementación del sistema, los registros se realizaban de forma manual, lo que generaba errores frecuentes, pérdida de información y demoras en la gestión de accesos.


Sistema web de control de accesos compuesto por módulos de administración, vigilancia, turnos, propietarios, inmuebles, personal externo, control de accesos y novedades. Permite la trazabilidad completa de ingresos y eventos, optimizando los procesos de seguridad del conjunto residencial.

 Aclaración importante
El conjunto residencial, los nombres y las imágenes utilizadas NO corresponden a un conjunto real. 
Las imágenes son únicamente de uso ilustrativo y académico.

Sin embargo, el proyecto  sí cumple con las funciones reales de un sistema de control de accesos**, simulando correctamente los procesos que se implementarían en un entorno real.


 Problemática principal
Antes del desarrollo del proyecto se presentaban las siguientes dificultades:

- Registros manuales con errores frecuentes  
- Dificultad para consultar historial de visitantes, vehículos y novedades  
- Pérdida de información o datos incompletos  
- Falta de control sobre vigilantes, propietarios y personal externo  
- Retrasos en los tiempos de respuesta del personal de seguridad  
- Vulnerabilidades en la seguridad por ausencia de información confiable  


 Solución propuesta
La aplicación Control de Accesos Conjunto Residencial Los Robles  permite:

- Eliminación de errores en los registros  
- Historial de accesos completo y de rápida consulta  
- Información segura y sin pérdidas  
- Mayor control del personal de seguridad  
- Respuesta más rápida ante novedades  
- Reducción de vulnerabilidades en el conjunto residencial  


Alcance del sistema
El sistema permite la gestión de los siguientes módulos:

- Administradores  
- Vigilantes  
- Turnos  
- Propietarios  
- Inmuebles  
- Personal externo  
- Control de accesos  
- Novedades  

Cada módulo permite:
- Registrar  
- Consultar  
- Actualizar  
- Eliminar  
- Listar
